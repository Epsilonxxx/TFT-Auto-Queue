# TFT-Auto-Queue

## 中文介绍

TFT-Auto-Queue 是一个适用于 LOL 国际服云顶之弈 的轻量桌面工具，主要用于 发条鸟试炼模式自动匹配。  
它可以自动处理重复操作，例如创建房间、开始匹配、接受对局，以及在对局结束后继续下一轮。

软件内置基础异常恢复能力，可在部分匹配失败、客户端重连异常、房间状态异常等场景下自动处理，尽量减少手动干预。  
同时支持 本地配置保存、中英文切换、F1 一键启动/关闭，适合长时间循环使用。

这是一个实验性质的 vibe coding 小白作品。  
主打一个简单直接、能跑就行、能省事就行。

## 主要功能

- 自动匹配发条鸟试炼
- 自动接受准备确认
- 对局结束后自动继续
- 匹配失败后自动冷却重试
- 连接异常时自动回房并重新匹配
- 客户端崩溃弹窗自动跳过恢复
- 循环次数统计
